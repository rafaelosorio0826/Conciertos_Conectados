Aplicación de Eventos Musicales
Descripción
Este proyecto es una plataforma web de eventos musicales donde los usuarios pueden explorar, comprar y gestionar entradas para conciertos y eventos musicales. La aplicación ofrece una experiencia intuitiva para descubrir eventos, acceder a detalles y comprar entradas de forma eficiente.

Problemática
La organización de eventos musicales enfrenta desafíos como falta de visibilidad, dificultad para acceder a información clara y un proceso de compra de entradas complicado. Esta aplicación busca resolver estos problemas mediante una interfaz accesible que facilite la búsqueda, visualización y compra de entradas.

Funcionalidades
Página de Inicio:

Buscador: Búsqueda de eventos.
Menú de Categorías: Filtros por tipo de evento.
Eventos de Tendencia: Muestra los eventos populares con detalles resumidos.
Eventos Cerca de Usted: Muestra eventos cercanos usando la ubicación del usuario.
Página de Detalles:

Cards de Presentación: Información básica del evento.
Descripción Completa: Detalles adicionales con opción de "ver más".
Valor y Localización: Ubicación, precio de entrada y botón de compra.
Página de Compra de Boleta:

Card de Evento: Información detallada sobre el evento, con código de barra y detalles del asiento.
